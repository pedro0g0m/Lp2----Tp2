Detailed Description

Classe Pessoa. Classe responsavel por gerar os objectos do tipo Pessoa e respeita o interface IPessoaModel. Tem os construtores e os varios metodos que manipulam as suas variaveis. A classe é Serializable para permitir que o objecto Pessoa possa ser armazanado num ficheiro.
