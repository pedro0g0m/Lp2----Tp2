Detailed Description

Classe ControlHistory. Esta classe serve para responder os pedidos reslizados pelo View invocando métodos declarados na camada Model. Desta maneira garantimos uma abstração há camada View, responsavel pela interação com o User, do funcionamento interna desta aplicação oferecendo mais segurança.
